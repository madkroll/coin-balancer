# Open Trade View

## Targets

1. Signals to buy source coin back
2. Signals to buy CG
3. Shows real-time win / loss

## Input Fields

- trade timestamp
- source coin
- source coin amount
- source CG price
- destination coin
- destination coin amount
- destination CG price

## Output controls

- baseline at trade timestamp
- trade date time
- minimum win level
- target win level
- loss red line
- real time update on client (1 query per minute)

## Signals

- send e-mails on crossing levels
